ЛАБОРАТОРНАЯ РАБОТА №2
по дисциплине: «Программирование»
на тему: «Классификация языков программирования»

Цели и задачи работы: изучение функций ввода-вывода данных, программирования вычисления значения выражения. 
Формирование практических навыков реализации алгоритмов на различных языках программирования 
(C++, Go (Golang), Swift, Python, а так же Ассемблер в номере 3), сравнительный анализ синтаксических и семантических особенностей языков.

Задание по варианту

1.	(Циклический сдвиг строк): Даны две строки S и T. Определите,
можно ли получить строку T циклическим сдвигом символов строки S.
Пример 1: S = “waterbottle”, T = “erbottlewat”, Результат: Yes
Пример 2: S = “abc”, T = “acb”, Результат: No
Пример 3: S = “ngtu”, T = “ngtu”, Результат: Yes

2.	(Пункты): Дан массив пар, содержащих начальный и конечный
пункты пребывания. Вернуть итоговый конечный пункт самой длинной цепочки следования.
Пример 1: ВВОД: ["Лондон", "Стамбул"], ["Новосибирск", "Лондон"],
["Стамбул", "Пекин"], ВЫВОД: “Пекин”
Пример 2: ВВОД: ["Барнаул", "Новосибирск"], ВЫВОД: “Новосибирск”
Пример 3: ВВОД: ["Искитим", "Искитим"], ВЫВОД: “Ошибка”
Пример 4: ВВОД: ["Москва", "Норильск"], ["Норильск", "Саратов"],
["Саратов", "Москва"], ВЫВОД: “Ошибка”

3.	На вход подаётся N натуральных чисел, необходимо посчитать
количество чётных цифр для каждого числа; если подаётся цифра (1,2ит.
4.	д.) – выводите знак -.
Пример 1: N = 4, 3 12 5 44, Результат: - 1 0 2
Пример 2: N = 3, 1 3 6, Результат: - - -
Пример 3: N = 4, 122 23 44 13, Результат: 2 1 2 0
